# Web UI

Web интерфейс встроен в backend и доступен по `/`.
Отдельного web‑проекта нет: это сознательное решение для демонстрационного scope.

## Камера (Web UI)

Камера запускается строго из пользовательского действия (клик/тач), чтобы корректно отображался системный диалог разрешений и гарантировалась совместимость с мобильными браузерами.

Основные инварианты:
- `getUserMedia()` вызывается синхронно внутри обработчика пользовательского жеста.
- `<video>` всегда находится в DOM и использует `autoplay`, `playsinline`, `muted`.
- При повторном запуске или ошибке все треки останавливаются, а `srcObject` очищается.
- Для переключения камеры используется `facingMode: { ideal: ... }`.

## Тесты Web UI
